# Portfolio Template — Building Your Portfolio in the AI Age

A template repo for building your own Quarto portfolio site.

## Quick Start

1. Click **"Use this template"** → **"Create new repo"**
2. Name it `<your-username>.github.io`
3. Launch a Codespace
4. Run `quarto preview` to see your site

## What's Included

- **Devcontainer** with Python, Quarto, and GitHub Copilot pre-configured
- **GitHub Actions** workflow for automatic deployment to GitHub Pages
- **Python** environment with common data science libraries
